# Alpha Construction


### `Mockup Design`

Simple mockup website for a construction company. 

Visit the site at: [Alpha Construction Website](https://alphaconstructionwebsite-lnsflive.vercel.app/).


## Built With
* NuxtJS (Front-End)
* Vuetify (Styling Framework)
* Strapi Headless CMS (Back-End)
